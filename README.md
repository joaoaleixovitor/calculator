# 📖 About

This is a simple calculator developed in Java with the purpose of practicing programming logic and reinforcing basic concepts of the language. The project performs simple arithmetic operations such as addition, subtraction, multiplication, and division.

It was created as a learning project to improve my Java skills and better understand the structure of a simple application.

- ✅ Basic mathematical operations
- ✅ Simple and easy-to-use interface
- ✅ Developed in Java
- ✅ Learning-focused project
- ✅ Programming logic practice

---

# 🎮 How to Use

1. Run the application.

2. The calculator window will open on the screen.

3. Click the buttons using your mouse to enter numbers and operations.

4. Choose a mathematical operation:

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`×`)
- Division (`÷`)

5. Press the equals (`=`) button to display the result.

6. Use the clear (`C`) button to reset the calculator.
